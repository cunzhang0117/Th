遊戲的故事主要是圍繞在一個家庭裡，玩家將扮演哥哥在家中探索，並透過提示紙條去找到生日禮物，藉由紙條的引導和周遭物品的探索，逐漸發現家中的異常以及真相。
<br>
以等距視圖為特色，呈現不一樣的視覺角度，讓玩家有更多面向的探索。
